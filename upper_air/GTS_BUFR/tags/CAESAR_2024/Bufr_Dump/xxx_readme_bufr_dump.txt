This directory contains a copy of all of the code and libraries needed for to run the bufr_dump code which converts the GTS BUFR data into an ASCII form. A copy of all of this should already be located on the machine at /opt/local/bin but just in case this copy has been created and placed in SVN. This is the version of bufr_dump from 2022 and used for SWEX data processing.
