How To Run the GTS BUFR Conversion Software     L. Cully Updated 22 Jan 2019
-------------------------------------------
1. Set up expected dir structure. Read the s/w headers to learn what is expected.

2. Copy raw, GTS BUFR (binary) data to proper input dir.

3. Run preprocess_GTS_BUFR.pl

   Run conversion to convert raw, BUFR binary to ascii and then to 
   preprocessed/cleaned ascii (e.g., preprocess_GTS_BUFR.pl).
   Note that xxx calls the bufr_dump executable (blackbox s/w) that
   converts the raw, GTS BUFR (binary) to expanded (and hard to read
   ascii).  The /lib/libeccodes.so required by the bufr_dump software.

    525053 Sep  4 10:57 bufr_dump
   7894726 Sep  4 15:05 libeccodes.so

   See the runit_BinToPreProc executable for the following execute statements for SOCRATES project:

   preprocess_GTS_BUFR.pl ../raw_gts_bufr_data/new_zealand ../output_ascii_data/new_zealand ../output_preproc_data/new_zealand >& preproc_NZ.log &
   preprocess_GTS_BUFR.pl ../raw_gts_bufr_data/antarctica ../output_ascii_data/antarctica ../output_preproc_data/antarctica >& preproc_ANT.log &
   preprocess_GTS_BUFR.pl ../raw_gts_bufr_data/australia ../output_ascii_data/australia ../output_preproc_data/australia >& preproc_AUS.log &


4. Run GTSBUFR_GetStats.pl
   Run Statistics software to gather stats and other info (e.g., GTSBUFR_GetStats.pl)
   on cleaned/preprocessed ascii data and provide to S. Loehrer. This runs on the *.preproc files
   generated by step 3. above. 

5. Run GTSBUFR_Radiosonde_Converter.pl
   Run conversion to convert cleaned/preprocessed ascii to ESC output format (e.g., GTSBUFR_Radiosonde_Converter.pl)

6. Run NWS sort software on the output *.cls from step 5. See the build.xml file.
Optional 7. Run format checker on sorted, ESC output data from step 6. See the build.xml file. This is Optional because step 8. when executed from the ant build.xml file will call check_format on the autoQC'd data files. 

8. Run autoQC s/w on ESC output data. This will also do check_format on the autoQC'd data files.

9. Have science staff, S.Loehrer, do Visual QC on output from step 8.

10. Make Day files that will be loaded into project archive.

11. If requested, generate 5mb form of QC'd ESC data.

12. If requested, generate 50 meter form of QC'd ESC data.
---end of steps-----
