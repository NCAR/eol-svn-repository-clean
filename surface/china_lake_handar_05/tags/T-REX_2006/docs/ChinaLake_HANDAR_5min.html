1.0  DATA SET OVERVIEW


This data set contains 5-minute resolution surface meteorological data
from the Naval Air Weapons Station (NAWS) China Lake network of 8 HANDAR 
weather stations.  These data were provided by NAWS China Lake for the 
T-REX period (March and April 2006).  Each file gzipped file contains 
one month of data for a station.  The data are in columnar ASCII.

3.0  DATA COLLECTION AND PROCESSING


NCAR/EOL conducted no processing or quality control on these data.


4.0  DATA FORMAT AND FILE NAMING

4.1  File Naming conventions

Files have names like:

0603_BAKER.txt.gz
where:
	06 is the two year
        03 is the two digit month
        BAKER is the station name (see Station Location section)


4.2  Data Format
 
These data are columnar ASCII.

The parameters are as follows:

Date
Time - Pacific Standard Time
Wind Direction - degrees
Wind Speed - knots
Wind Gust - knots
Temperature - degrees F
Relative Humidity - percent
Station Pressure - mb
Battery Voltage
